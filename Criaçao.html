Desenvolvimento do Projeto

Base do projeto

A base do projeto foi desenvolvida utilizando duas tábuas de madeira, com o objetivo de melhorar a sustentação do protótipo anterior e criar uma plataforma mais resistente e ampla, preparada para impressões maiores e mais desafiadoras.

O mecanismo principal do projeto é composto por dois motores de passo. Eles são o coração do sistema de movimentação, sendo os únicos responsáveis por controlar o deslocamento preciso nos eixos X e Y. Através de comandos eletrônicos, estes motores garantirão que a ferramenta de trabalho se mova de forma exata e coordenada para executar as tarefas.

A imagem ilustra a montagem da estrutura principal do projeto, onde os dois mecanismos com motores de passo serão fixados na base de madeira. Um atuador será posicionado na horizontal para formar o eixo X, enquanto o segundo será montado na vertical, perpendicularmente ao primeiro, para criar o eixo Y. Esta configuração em ângulo reto é o "esqueleto" da máquina, permitindo o movimento em duas dimensões e formando a área de onde o projeto irá operar.

Projeto funcional

O vídeo mostra os mecanismos sendo testados antes de iniciar a fixação ao suporte. Este passo foi fundamental para verificar se cada motor respondia corretamente aos comandos do microcontrolador e se movia por todo o seu eixo corretamente. Ao finalizar a verficação foi possível avançar para a montagem final com a certeza de que os componentes principais estavam funcionais.

Para a montagem final, os mecanismos foram fixados ao suporte de madeira utilizando hastes de metal, parafusos e porcas. Esse método de fixação garantiu que ambos os eixos ficassem alinhados, eliminando folgas ou vibrações. Nesta mesma etapa, foi integrado o suporte da caneta e seu sistema de controle, responsável pelo movimento vertical de encostar e levantar a ponta do papel durante o desenho.

Projeto finalizado e funcional, realizando a impressão de um desenho para fins de testes. Este teste serviu para confirmar que todos os componentes a estrutura, os motores e o sistema de controle estavam funcionando. O resultado demonstrou uma grande melhoria na precisão dos movimentos e na velocidade de impressão com relção ao primeiro protótipo, concluindo com sucesso os objetivos iniciais do projeto.
